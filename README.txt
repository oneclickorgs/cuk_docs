This is internal documentation for Co-operatives UK.

It is designed to be viewed as a Gollum-powered wiki:

$ gem install bundler
$ bundle install
$ bundle exec gollum

then visit http://localhost:4567/ in your browser.

To generate a static version of the site, use gollum-site:

$ gem install gollum-site
$ gollum-site generate

The generated static files will be placed in the '_site' directory.
